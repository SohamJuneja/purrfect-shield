# 🛡️ Purrfect Shield

**A TEE-Secured Contract Forensics Skill for Purrfect Claw Agents**

Purrfect Shield is a Web3 security skill built for the **Pieverse Skill Store**. It solves the biggest problem with autonomous Web3 agents: *blindly signing malicious contracts.* By acting as a middleware interceptor, Purrfect Shield analyzes BNB Smart Chain token contracts for honeypots, exorbitant taxes, and malicious proxy logic before allowing the agent's TEE wallet to broadcast the transaction.

## 🏆 Bounty Alignment: "Build Web3 Skills for Purrfect Claw Agents"
- **Native TEE Integration:** Directly integrates with the `purr` CLI to execute safe transactions from within the hardware-isolated environment.
- **On-Chain Action:** Automates the `purr evm send` workflow natively.
- **Real-World Value:** Protects autonomous trading agents from being drained by malicious Meme tokens on Four.meme.
